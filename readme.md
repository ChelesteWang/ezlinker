## 说明
本项目是一个非常根正面红的Springboot项目，对于想开发的朋友来说很方便使用。同时编译起来也很简单：
```shell script
mvn clean
mvn package
```
最后在app下有个target目录，同时会打包一个Docker镜像。
> 哦，差点忘了，需要提前装好Docker和数据库等等。
